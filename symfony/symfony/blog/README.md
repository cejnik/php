# Blog

This is a small Symfony learning project. The goal was to practice basic routing, controllers, Twig templates, POST forms, redirects, and storing simple data in the session.

The application allows adding blog posts, listing them, and deleting them.

## Main Features

- Blog post list
- Add new post
- Delete post
- Session-based storage
- Basic Twig templates

## Technologies

- PHP
- Symfony
- Twig
- HTML
- Session storage

## What I Practiced

- Symfony routes and controllers
- Reading form data from `Request`
- Handling POST requests
- Redirecting after form submission
- Passing data from controller to Twig
- Using session as temporary storage

## Setup

```bash
composer install
symfony serve
```

## Note

This project is only a learning exercise. It does not use a database.
