# Task Manager

This is a small Symfony learning project focused on basic task management with session storage.

The application allows adding simple notes/tasks, displaying them, and deleting them.

## Main Features

- Task list
- Add task
- Delete task
- Session-based storage
- Basic Twig templates

## Technologies

- PHP
- Symfony
- Twig
- HTML
- Session storage

## What I Practiced

- Symfony routes and controllers
- Form submission with POST
- Working with `Request`
- Working with session data
- Rendering lists in Twig
- Basic delete flow

## Setup

```bash
composer install
symfony serve
```

## Note

This project is a simple practice app and does not use Doctrine or a database.
