# CookingBook

This is a Symfony learning project built after the blog and task manager exercises. The goal was to repeat session-based CRUD logic and add editing.

The application allows managing a small recipe list stored in the session.

## Main Features

- Recipe list
- Add recipe
- Edit recipe
- Delete recipe
- Session-based storage
- Basic page-specific styling

## Technologies

- PHP
- Symfony
- Twig
- HTML
- CSS
- Session storage

## What I Practiced

- Symfony controller actions
- GET and POST request flow
- Manual HTML forms
- Editing existing items
- Deleting items with POST forms
- Passing route parameters
- Reusing the same data structure across templates

## Setup

```bash
composer install
symfony serve
```

## Note

This project is intentionally simple and uses session storage instead of a database.
